Задание 1

  1. Запустите Microsoft Excel 2019.
  2. Откройте файл exercise_02_1.xlsx.
  3. Откройте файл exercise_02_2.xls.
  4. Откройте файл базы данных Microsoft Office Access 2007 exercise_02.accdb. Для открытия выберите таблицу Май.
  5. Создайте новую пустую книгу.
  6. Создайте новую книгу на основе шаблона Отчет о продажах.
  7. Закройте все открытые файлы. Изменения не сохраняйте. Закройте Microsoft Excel 2019.

Задание 2

  1. Запустите Microsoft Excel 2019.
  2. Откройте файл exercise_02_1.xlsx. Сохраните файл под именем Lesson_02_1.
  3. Откройте файл exercise_02_2.xls. Сохраните файл в формате Книга Excel (Excel 2019) под именем Lesson_02_2.xlsx.
  4. Откройте файл exercise_02_3.xlsx. Сохраните файл в формате Книга Excel 97-2003 под именем Lesson_02_3.xls.
  5. Откройте файл exercise_02_4.xls. Преобразуйте файл в формат Книга Excel (Excel 2019).
  6. Закройте все открытые документы. Изменения не сохраняйте. Закройте Microsoft Excel.

Задание 3

  1. Запустите Microsoft Excel 2019.
  2. Откройте файлы exercise_02_5.xlsx, exercise_02_6.xlsx, exercise_02_7.xlsx.
  3. Установите одновременное отображение файлов exercise_02_5.xlsx, exercise_02_6.xlsx с возможностью синхронной прокрутки листов.
  4. Отмените одновременное отображение файлов exercise_02_5.xlsx, exercise_02_6.xlsx.
  5. Установите одновременное отображение всех открытых файлов.
  6. Закройте все открытые документы. Изменения не сохраняйте. Закройте Microsoft Excel 2019.
