Создание документов

 

Новый документ создается автоматически сразу после запуска Excel.

Новый пустой документ или документ на основе шаблона можно создать в процессе работы.

  1. Нажмите вкладку Файл и выберите вкладку Создать (см. рис. 2.9).
  2. В появившейся области (рис. 2.12) для создания пустого документа выберите Пустая книга.

  1. Для создания документа на основе шаблона выберите его в этом же окне (на рис. 2.13 шаблоны отсутствуют).

Рис. 2.13. Создание документа на основе установленного шаблона

  1. При наличии подключения к Интернету можно обратиться к шаблонам, расположенным на сайте Microsoft Office Online. Выберите соответствующую группу шаблонов, дождитесь, пока они загрузятся. Прокручивая список (рис. 2.14) выберите нужный шаблон и дважды щелкните по его значку.

При нажатии на шаблон откроется окно с информацией о шаблоне (рис. 2.15). Чтобы создать файл с этим шаблоном, нажмите кнопку Создать.

Сохранение файлов

Сохранение изменений в существующем файле

Для сохранения изменений в существующем файле достаточно нажать кнопку Сохранить панели быстрого доступа.

Сохранение изменений в виде нового файла

Для сохранения нового документа в виде файла или существующего документа в виде нового файла (с другим именем и/или в другой папке) выполните следующие действия.

  1. Нажмите вкладку Файл и выберите вкладку Сохранить как (см. рис. 2.9), затем нажмите кнопку Обзор.
  2. В окне Сохранение документа (см. например, рис. 2.7) перейдите к нужной папке.
  3. В поле Имя файла введите (при необходимости) имя файла (расширение имени вводить не следует) и нажмите кнопку Сохранить.
