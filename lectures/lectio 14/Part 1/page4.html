Добавление и удаление элементов диаграммы

Выбор макета диаграммы

Макет диаграммы определяет наличие и расположение элементов диаграммы.

  1. В группе Макеты диаграмм вкладки Работа с диаграммами/Конструктор щелкните по кнопке Экспресс-макет.
  2. Выберите требуемый вариант из галереи макетов (рис. 14.11).

    3. Рис. 14.11. Выбор макета диаграммы

Содержание галереи макетов зависит от типа диаграммы.

Добавление и удаление элементов диаграммы

Независимо от выбранного макета диаграммы можно добавлять и удалять ее отдельные элементы. Для этого используют элементы выпадающего списка при нажатии на кнопку Добавить элемент диаграммы вкладки Работа с диаграммами/Конструктор (рис. 14.12).

Рис. 14.12. Добавление и удаление элементов диаграммы

Например, можно добавить название диаграммы.

  1. Щелкните по кнопке Добавить элемент диаграммы вкладки Работа с диаграммами/Конструктор.
  2. В выпадающем списке выберите пункт Название диаграммы.
  3. Выберите вариант расположения названия диаграммы (рис. 14.13).

    4. Рис. 14.13. Выбор расположения названия диаграммы

  4. В поле названия диаграммы с клавиатуры введите название диаграммы (рис. 14.14).

    5. Рис. 14.14. Добавление названия диаграммы

Для удаления названия диаграммы в меню пункта Название диаграммы (см. рис. 14.13) выберите вариант Нет.

Например, можно добавить на диаграмму легенду.

  1. Щелкните по кнопке Добавить элемент диаграммы вкладки Работа с диаграммами/Конструктор.
  2. В выпадающем списке выберите пункт Легенда.
  3. Выберите вариант расположения легенды (рис. 14.15).

    4. Рис. 14.15. Добавление легенды

Для удаления легенды в меню пункта Легенда (см. рис. 14.15) выберите вариант Нет.

Аналогично можно добавить и удалить и другие элементы диаграммы.

Кроме того, для удаления любого элемента диаграммы можно выделить его и нажать клавишу Delete.
